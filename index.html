Live bus timetable

Live bus timetable are connected with the GPS of each bus. They also communicate through 3G so the live bus timetable board can get the latest information for example bus being cancelled or delayed.

Spark Wifi Zone

The spark wifi zone is a part of the spark payphone service. It provides wifi within a distance of 50m. Users can choose from three wifi plans which are: Guest, Spark Broadband user and Spark mobile plan user.
